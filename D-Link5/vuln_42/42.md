## Vulnerability description

We found an Command Injection vulnerability  in D-Link Technology router with firmware which was released recentlyï¼Œallows remote attackers to execute arbitrary OS commands from a crafted request.

**Remote Command Execution**

In `alphapd` binary:

In `setSystemAdmin` function, `AdminID` is directly passed by the attacker, so we can control the `AdminID` to attack the OS.
